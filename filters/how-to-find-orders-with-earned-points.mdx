---
title: "How to find orders with earned points"
---

Go to **Customer Data Platform** → **Actions**.
​

![](/images/imported/How-to-find-orders-with-available-earned-points.png)

**Find orders with earned points that are blocked. **
​

![](/images/imported/How-to-find-orders-with-available-earned-points-(1).png)

**Find orders with earned points that are available.**

![](/images/imported/How-to-find-orders-with-available-earned-points-(2).png)

**Find orders with earned points that are fully spent.**

![](/images/imported/How-to-find-orders-with-available-earned-points-(3).png)

Replace the **Order** filter with **“Order First Action = Yes”** to filter only the first order event per order. Filtering by the first event (for example *Order Placed*) ensures there is exactly one unique event per order, which guarantees accurate order counts in your filter.
You can find more details on **Order First Action = Yes filter **in [this article](https://help.maestra.io/en/articles/13667784-how-the-first-order-action-filter-works).

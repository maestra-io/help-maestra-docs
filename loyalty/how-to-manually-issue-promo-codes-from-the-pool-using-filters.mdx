---
title: "How to Manually Issue Promo Codes from the Pool Using Filters"
---

You can issue single-use promo codes from the [pool of promo codes](https://intercom.help/maestraio/en/articles/11899917-how-to-import-promo-codes-into-a-project) to your customers automatically (at a certain step of a flow or an API method) or manually (to customers that you’ve filtered or customers listed in a CSV file).

# To assign promo codes manually, follow these steps:

1. Build your filter in the **Customers** section:

![00_issuecode](/images/imported/00_issuecode.png)

2. Open the drop-down menu and select the relevant customers or check the boxes manually:

![01_issuecode](/images/imported/01_issuecode.png)

You can click **Select all** or **Random selection**. Example: 500 customers or 20% of all the customers found.

3. Open the menu and select **Issue promo code**:

![02_issuecode](/images/imported/02_issuecode.png)

4. Select the promo code pool and click **Apply**:

![03_issuecode](/images/imported/03_issuecode.png)

You can only select pools that are marked as "Can be used" in the system.

5. A link to the task appears:

![04_issuecode](/images/imported/04_issuecode.png)

6. Please wait until this task is completed:

![05_issuecode](/images/imported/05_issuecode.png)

Once the task has been completed successfully, the relevant action will be recorded for the customer:

![06_issuecode](/images/imported/06_issuecode.png)

And promo codes will be assigned:

![07_issuecode](/images/imported/07_issuecode.png)

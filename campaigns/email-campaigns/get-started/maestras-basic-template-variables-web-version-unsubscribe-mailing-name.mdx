---
title: "Maestra’s Basic Template Variables (Web Version, Unsubscribe, Mailing Name)"
---

Transliterated name of your message:

```
`${Message.MailingUtmName}`
```

Web version variable (the "Open in Browser" CTA):

```
`${Message.AccessibilityLink}`
```

Sending date & time:

```
`${Message.SendingDateTime}`
```

Channel unsubscribe link:

```
`${Message.UnsubscribeLink}`
```

Topic unsubscribe link:

```
`${Message.TopicUnsubscribeLink}`
```

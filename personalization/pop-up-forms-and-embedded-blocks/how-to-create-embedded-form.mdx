## How to Add a Form to Your Website
**Placement Settings**:
- Embed content block into page selector:
  - Embed form placement: before/after element, at the start/at the end of the block, replace content of the block, replace block
  - Selector ‚Äã
  ![](/images/imported/--2026-01-29--14_20_52.png)
- Place content block manually Copy the code and paste it into the appropriate location in your website template. ‚Äã
  ![](/images/imported/--2026-01-29--14_22_34.png)**How to Choose a Selector** ‚ÄãA **CSS selector** is the element to which specific CSS properties will be applied.
1. Install the **[Selector Gadget](https://chromewebstore.google.com/detail/selectorgadget/mhjhnkcfbdhnjickkkdbjoemdmbfginb)** extension for Google Chrome. It significantly simplifies finding the correct selector.
2. Go to the website where you want to place the form and choose the location on the page where it should appear.
3. Activate Selector Gadget, hover your cursor over the desired area, and copy the selector.Common selector types:
- **`.someClass`** ‚Äî targets any element with the class `someClass`
- **`#someId`** ‚Äî targets the element with the  `someId` ID.You can find a detailed guide on selecting the right selector in the article. ‚ÄãIf you need to use different selectors for desktop and mobile for the same form, use the following in the selector field:

```
`.popmechanic-desktop [selector_for_desktop], .popmechanic-mobile [selector_for_mobile]`
```
where you need to replace `[selector_for_desktop]` and `[selector_for_mobile]` with the corresponding selectors.

**To Create a pop-up:**

1. Go to **Campaigns Manager→ Create campaign**
2. **Personalization → Pop-up**.

![](/images/imported/How-to-Create-Pop-ups.png)

Choose a folder and click **Create**.

![](/images/imported/How-to-Create-Pop-ups-(1).png)

<Warning>

**To work with product list data and browsed products, full integration is required.**

Follow the instructions to configure the settings: [https://developers.maestra.io/page/website-personalization](https://developers.maestra.io/page/website-personalization)
</Warning>

## **Choose a template**

Next, choose a template depending on the purpose of the pop-up. You can use it to:

- Generate leads
- Increase conversion & average order value
- Notify visitors
- Conduct a survey

![](/images/imported/How-to-Create-Pop-ups-(3).png)

After choosing a template, specify your website and click the **Proceed** button.

![](/images/imported/How-to-Create-Pop-ups--2819-29.png)

## **Edit the template**

Once the pop-up is created, click the **Edit** button **.**

![](/images/imported/How-to-Create-Pop-ups--2820-29.png)

On the Pop-up editing page click the **Edit** button. 

![](/images/imported/How-to-Create-Pop-ups-(21).png)

The form editing window is displayed below.

![](/images/imported/How-to-Create-Pop-ups-(22).png)

**Here you can:**

- Adjust the pop-up’s **position** on the website (Form position **button)**
- Check how the pop-up **displays on** **different devices (Desktop/Tablet/Phone)**.
- Configure **background color, icons, border and field sizes, fonts, privacy policy** and other visual options. 

![](/images/imported/How-to-Create-Pop-ups-(23).png)

- Edit the pop-up directly in HTML code (&lt;&gt;Code button)
- Configure the main screen and the confirmation screen after the customer completes the action.
- Preview the form on your website. ​

## **New pop-up templates**
New templates allow you to create forms that are different for each device. They are also easier and more intuitive to configure. You can find them in the gallery, where they are marked with a "New"  label.

  ![](/images/imported/How-to-Create-Pop-ups-(24).png)

- 

### Pop-up editing pageWhen you open a page, you’ll see a list of elements included in the form. You can move, delete, or add new ones. 

  ![](/images/imported/How-to-Create-Pop-ups-(25).png)
In addition to basic input fields, new templates also support adding:
- timer
- promo code field
- option buttons
- close button
  ![](/images/imported/How-to-Create-Pop-ups-(26).png)

- 

### **Editing elements**To edit an element, click it in the list or on the preview.

  ![](/images/imported/How-to-Create-Pop-ups-(27).png)

<Note>
New templates allow you to customize each text element for different screens, including font style, size, color, and alignment.
</Note>

- 

### **Device-specific design**By default, most elements automatically sync and adjust to the mobile version.To set custom properties, disable synchronization and configure settings for the specific device.

  ![](/images/imported/How-to-Create-Pop-ups-(28).png)
- 

### **General form settings**To adjust general form settings (font, close button, layout, etc.), go back to the General settings page. 

  ![](/images/imported/How-to-Create-Pop-ups-(29).png)

  ![](/images/imported/How-to-Create-Pop-ups-(30).png)

<Note>
With the new templates, you can set a fixed height for all screens. This is useful when screens contain different content and their heights vary.
</Note>

- 

### **Saving data when closing the second screen**For templates with multiple main screens, you can configure the form to save entered data when the second screen is closed.In this case, all fields on the second screen become optional.
To enable: open **General settings desktop→ Data transfer → User data** and activate the option.

  ![](/images/imported/How-to-Create-Pop-ups-(31).png)
We only recommend using this option if the first screen **includes a mandatory privacy policy checkbox** to ensure that the user has explicitly consented to data transfer and communications.

Use filters to define who will see the pop-up and under what conditions.You can preconfigure a customer segment and show (or hide) the form for that segment.

## General Settings
In **General settings**, specify:
- Website
- Days of the week for display
- Frequency of appearance
- Whether the form is **priority**. If checked, the pop-up will appear even if frequency limits per visitor are set in the “Pop-up frequency control” section.

<Note>
For example, these are the settings for a form that will be displayed daily, but will only pop up once for each user during a visit to the site (and form is a priority).

  ![](/images/imported/How-to-Create-Pop-ups-(33).png)
</Note>

## **Display conditions**
Next, set the pop-up display conditions: ​1.  By set of conditions
- After Exit intent (cursor leaves the active page area) - only for desktop campaigns.
- After page scrolled for X %
- After a certain time spent on the page.
- After visiting a defined number of pages.
- After being inactive for period of time
- On a specific action performed on the website.2. Immidiately after falling into target group 3. After API-request or click on call-to-action button

<Note>
For example, these are the settings for a form that will be displayed for a customer who has visited two pages of the site in total.

  ![](/images/imported/How-to-Create-Pop-ups-(34).png)
</Note>

<Warning>
Page views and visits are counted starting from the moment the popup script is installed on the site. If installed yesterday, data is only available from that point.
</Warning>

Click Form preview. Insert your website link and open it. 

  ![](/images/imported/How-to-Create-Pop-ups-(36).png)
You will then see the pop-up live on your site.

  ![](/images/imported/How-to-Create-Pop-ups-(37).png)

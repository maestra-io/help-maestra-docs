---
title: "How to Set Up the Ad Optimization Module"
---

The Ad Optimization module automatically transmits audience segment data from Maestra to ad dashboards such as:

- Google Ads;
- Facebook;
- MyTarget.

The list of ads dashboards will be extended as new ads dashboards are released.

# How to create a new Ad cabinet integration

1. Go to **Integration Settings** → **Integration Setup** and click the "**Create integration**" button:

![](/images/imported/2025-12-05_12-56-06.png)

2. Then choose to create Advertising cabinet:
​

![](/images/imported/2025-12-05_13-15-17.png)

3. Name the **Ad cabinet integration**, choose a brand (for multi brand projects) and click the "**Create**" button:
​

![](/images/imported/2025-12-05_13-02-53.png)

# How to add connections to ads dashboards

<Note>

Enable module on **Billing detalization page** in **Administrative settings** on your own.
</Note>

1. Go to **Integration**→ **Ad Optimization** and open the Connections tab:

![03_adoptimize](/images/imported/03_adoptimize.png)

2. Select a relevant ads dashboard:

![04_adoptimize](/images/imported/04_adoptimize.png)

3. Name your connection:

![05_adoptimize](/images/imported/05_adoptimize.png)

4. Click **Grant access**:

![06_adoptimize](/images/imported/06_adoptimize.png)

The connection has been added successfully!

# How to create an integration

1. Go to **Apps** and click **New**:

![07_adoptimize](/images/imported/07_adoptimize.png)

2. Select the source of data import — Maestra:

![08_adoptimize](/images/imported/08_adoptimize.png)

3. Always check in **real time** for your integration operation:

![10_adoptimize](/images/imported/10_adoptimize.png)

4. Select your method of data encryption and the audience segment you want to send from Maestra:

![11_adoptimize](/images/imported/11_adoptimize.png)

<Note>

**How to create a segment**
Refer to the guidelines on [How to create a segment.](/segments/how-to-filter-customers-into-segments)
</Note>

5. Select where to send data to:

![12_adoptimize](/images/imported/12_adoptimize.png)

6. Set up the fields you want to transmit:

![14_adoptimize](/images/imported/14_adoptimize.png)

![15_adoptimize](/images/imported/15_adoptimize.png)

7. Click **Start** to launch the integration:

![16_adoptimize](/images/imported/16_adoptimize.png)

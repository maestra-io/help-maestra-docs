title: "How to Bulk Import Customers from a File?"
---

## **1. Prepare your file for import**

Go to **Database** → **Customers** → **Import .csv** → **Customer data:**

![00_importcustomer](/images/imported/img-2c2357c1.png)

Select **Customer import** from the **Task** drop-down menu:

![01_importcustomer](/images/imported/img-aab31e79.png)

Now you’ll need to prepare a file for import.

The supported format is CSV. Use this guide on field names to help get your file ready and ensure there aren’t any errors.

Here’s an example of a file that’s ready for import:

![02_importcustomer](/images/imported/img-d1b6ebe0.png)

## **2. Import customers**

## **2.1. Fill in the settings:**

**Task:** Customer import.

**Description:** fill in if applicable. This description can help you find your import task in the tasks list.

**Encoding:** the default encoding is UTF-8.

**Uploaded file:** follow Step 1 to upload the import file.

![03_importcustomer](/images/imported/img-63eef99d.png)

## **Import settings:**

- **API endpoint:** what you choose here depends on your task — but in general, when importing data manually via the interface, selecting Maestra here makes the most sense.
- **Brand:** for multibrand projects.
- **Touchpoint:** here, you can simply check "define automatically" and the system will take care of the rest.
- **Source action:** you can use the default action, select other action from the drop-down menu or use value from the imported file.Source action will be added to new customers' activity feeds.If you [create an action template](/customers-orders-and-products/actions/how-to-create-an-action-template), make sure the **action name** is easy for you to find – for example "Filled in May "22 survey" or "Registered on xyz.com".
- **Action for existing customers:** use the default action or another suitable action. This action will be added to existing customers' activity feeds.

## **2.2. Click Launch import to add your task:**

![05_importcustomer](/images/imported/img-2bd56646.png)

Now you can see a URL leading to this task:

![06_importcustomer](/images/imported/img-fdb6fd4c.png)

Wait for the import to be completed:

![07_importcustomer](/images/imported/img-0faffb93.png)

The customer will emerge in the database:

![08_importcustomer](/images/imported/img-36c4c597.png)

To open the profile of the customer, click the customer's name:

![09_importcustomer](/images/imported/img-6d0a0ddf.png)

---
title: "How to Add a Customer Manually via the User Interface"
---

To add a customer using the Maestra interface, follow these steps:

1. Go to **Database** → **Customers** and click **Add customer** in the side menu:

![00_custcreate](/images/imported/00_custcreate.png)

2. Fill in the fields.

**IDs**: at least one contact field must be filled in:

![1_custcreate](/images/imported/1_custcreate.png)

**Personal data**: this section is optional.

![2_custcreate](/images/imported/2_custcreate.png)

Specify your **Subscription preferences**. By default, the subscription fields are blank. You can select the required status from the list:

![3_custcreate](/images/imported/3_custcreate.png)

[Custom fields:](/api-integrations/how-to-create-a-custom-field) this section is optional:

![4_custcreate](/images/imported/4_custcreate.png)

3. Click **Create**:

![5_custcreate](/images/imported/5_custcreate.png)

title: "How to Add a Customer Manually via the User Interface"
---

To add a customer using the Maestra interface, follow these steps:

1. Go to **Customer Data Platform** → **Customers**.

<Frame>
  ![2026 05 08 19 51 09](/images/2026-05-08-19.51.09.png)
</Frame>

2. Click **Create customer** in the side menu.

<Frame>
  ![Снимок Экрана — 2026 05 08 В 19 55 55](/images/Снимок-экрана-—-2026-05-08-в-19.55.55.png)
</Frame>

3. Fill in the fields.

- **IDs**: at least one contact field must be filled in:

![1\_custcreate](/images/imported/1_custcreate.png)

- **Personal details**: this section is optional.

<Frame>
  ![2026 05 08 20 02 45](/images/2026-05-08-20.02.45.png)
</Frame>

Specify your **Subscription preferences**. By default, the subscription fields are blank. You can select the required status from the list:

![3\_custcreate](/images/imported/3_custcreate.png)

[Custom fields:](/api-integrations/how-to-create-a-custom-field) this section is optional:

![4\_custcreate](/images/imported/4_custcreate.png)

3. Click **Create**:

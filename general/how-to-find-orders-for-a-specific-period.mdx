---
title: "How to find orders for a specific period"
---

- Go to **Customer Data Platform→ Actions.**

![](/images/imported/How-to-find-orders-for-a-specific-period.png)

- The **Actions** page shows all actions from all customers. First, let's filter order-related actions only. This filter shows **all** order actions. 

![](/images/imported/How-to-find-orders-for-a-specific-period-(1).png)

- To get an accurate order count, make sure you only count each order once. Orders can go through multiple statuses (Placed → Paid → Delivered), but you should filter by the first order action so you’re counting the order itself, not each status update. ​
  ![](/images/imported/How-to-find-orders-for-a-specific-period-(2).png)
- Select the required order statuses.
  ![](/images/imported/How-to-find-orders-for-a-specific-period-(3).png)

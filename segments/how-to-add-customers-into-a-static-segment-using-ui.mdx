---
title: "How to Add Customers into a Static Segment using UI"
---

Follow the guidelines to filter customers into a Static segment.

Refer to *How to Import Customers into a Static Segment* to learn how to upload customers into a segment from a file.

---

## Steps

1. Check if the segmentation is active:

![](/images/imported/--2025-07-30--4_22_30PM.png)

Refer to [How to create a static segment](/segments/how-to-create-a-static-customer-segment).

2. Build your filter in the **Customers** section and click **Apply**:

![](/images/imported/--2025-07-30--4_26_39PM.png)

3. Open the drop-down menu and select the relevant customers or check the boxes manually:

![](/images/imported/--2025-07-30--4_29_42PM.png)

You can click **Select** all or **Random selection**. Example: 500 customers or 20% of all the customers found.

4. Click **Selected** -> **Add to segment**:

![](/images/imported/--2025-07-30--4_32_45PM.png)

5. Select the **Segmentation** and the **Segment.** You will have the segmentation as the only option if it contains just one segment.

Click **Add**:

![](/images/imported/--2025-07-30--4_35_36PM.png)

6. A link to the task appears:

![](/images/imported/--2025-07-30--4_38_39PM.png)

7. Please wait until this task is completed:

![](/images/imported/--2025-07-30--4_41_11PM.png)

8. Your segment will appear in the customer profile on the **Segments** tab:

![](/images/imported/--2025-07-30--4_43_00PM.png)
